# AI Financial Report Generator using LLMs

This project builds an AI-powered financial analysis pipeline that converts raw financial datasets into automated financial reports.

The system calculates financial metrics, generates visual dashboards, and produces an AI-generated audit-style report using a Large Language Model.

## Features

- Financial metric calculation
- Revenue growth analysis
- Debt ratio evaluation
- Risk flagging
- AI-generated financial summary
- Financial dashboard generation

## Tech Stack

- Python
- Pandas
- Matplotlib
- Seaborn
- Generative AI (OpenAI / Llama)

## Project Workflow

CSV Financial Data  
↓  
Python Analysis  
↓  
Financial Metrics Extraction  
↓  
LLM Generated Financial Report  
↓  
Dashboard Visualization
