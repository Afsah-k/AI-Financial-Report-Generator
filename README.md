# AI Financial Report Generator using LLMs

This project uses Python and Large Language Models to analyze financial data, calculate key metrics, generate a dashboard, and produce an AI-written financial report.



## Features

* Financial metric calculation
* Revenue growth analysis
* Debt ratio and risk flagging
* AI-generated financial summary
* Visual dashboard generation



## Tech Stack

* Python
* Pandas
* Matplotlib
* Seaborn
* OpenAI / Llama



## Project Workflow

CSV Financial Data

↓

Python Analysis

↓

Financial Metrics Extraction

↓

LLM Generated Financial Report

↓

Dashboard Visualization



## How to Run

1. Install dependencies
2. Run the Python script
3. View the generated dashboard and markdown report



## Example Outputs

The project generates:

* Financial dashboard image
* AI-generated financial report in Markdown format
